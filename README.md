UFC Fight Tracker

- Fight Countdown
- Fight Location
- Where to watch fight
- Fight #
- Fighters participating
- Fighter ranking
- Clicking on name takes you to fighter UFC profile
- Time of Fight
- Weight Class
- If fighter is champ
- Main Card Only toggle

  
- Past fights
  - Winner
  - Method of Victory
  - Round Time
  - Round #
